# aliddns - 域名解析设置工具(阿里云)

## 使用说明：
>1、填入阿里云的**AccessKey lD**和**AccessKey Secret**

>2、点击**<保存并加载>**按钮，即可加载所有在阿里云添加的域名

>**3、账号数据以明文方式存储在程序所在目录AccessKey.json文件中，请妥善保存数据**

## 解析支持说明：
>支持IPV4

>支持IPV6
